# 🌾 Grassland Dimension Mod - 使用说明

## 📦 模组文件

```
C:\Users\soeon\Desktop\void-space-template-1.21.1\build\libs\void-space-1.0.0.jar
```

## 🎮 如何使用

### 1. 安装模组

1. 打开 Minecraft mods 文件夹：
   ```
   %APPDATA%\.minecraft\mods
   ```

2. 将 `void-space-1.0.0.jar` 复制到 mods 文件夹

3. 启动 Minecraft（使用 Fabric Loader）

### 2. 进入维度

**方法 1：使用指令**
```
/grassland
```

**方法 2：手动创建维度**

如果指令不工作，你需要手动创建维度：

1. 在 Minecraft 中创建新世界
2. 进入世界后，打开 `saves\<世界名>\datapacks` 文件夹
3. 创建一个数据包，添加维度定义

### 3. 维度特性

- 🌾 **全是草方块** - 超平坦地形
- ☀️ **永远白天** - 没有黑夜
- 🌤️ **无天气** - 没有雨雪
- 📍 **坐标 (0, 65, 0)** - 传送位置

## 📝 注意事项

- 这是一个基础版本的模组
- 维度功能需要通过数据包或指令来完全实现
- 如果需要完整的维度功能，需要添加更多的 Java 代码

## 🔧 故障排除

### 问题：模组加载失败
- 确保使用了 Fabric Loader
- 检查 Minecraft 版本是否为 1.21.1

### 问题：指令不工作
- 确保你有管理员权限（OP）
- 尝试手动创建维度
